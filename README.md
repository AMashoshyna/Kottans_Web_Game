# KottansGame

This repository is a pet project for Kottans Web Course 2017!


## How to begin

1. Install [Docker](https://docs.docker.com/engine/installation/) and [docker-compose](https://docs.docker.com/compose/install/)
2. docker-compose build
3. docker-compose up
